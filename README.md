# R1AMOBI - Conectando Destinos 🏍️💨 🏁📦

Central digital oficial da **R1AMOBI** desenvolvida para facilitar o acesso de clientes, parceiros e passageiros aos nossos serviços de mobilidade, entregas e tecnologia em Assis-SP.

## 🚀 Funcionalidades Integradas
* 📲 **Instalação Direta (PWA):** O site pode ser instalado direto na tela inicial do smartphone como um aplicativo próprio, com ícone oficial e tela cheia.
* 📊 **Calculadora de Orçamento Particular:** Sistema inteligente integrado para simulação de corridas urbanas, entregas e viagens intermunicipais com envio automático de rotas mapeadas para o WhatsApp.
* 🛍️ **Carrossel de Anúncios Ativo:** Vitrine digital no rodapé integrada com a nossa loja de equipamentos e marketplace da marca.
* 🎵 **Playlist Exclusiva:** Player conectado diretamente à nossa playlist oficial no YouTube.

## 🛠️ Tecnologias Utilizadas
* HTML5 / CSS3 (Design Moderno Cyberpunk com efeitos Neon)
* JavaScript Vanilla (Motor de partículas para efeito de asfalto e lógica da calculadora)
* Service Workers & Web App Manifest (Conversão em Progressive Web App)
